This is a team project for AI 570, using FER 2013 dataset to classify emotions.

1. Goal: Able to correctly identify emotions with light models. 

2. Challenge: data with low recogition, imbalanced classes

3. Solution: Try using Data augmentation, dropout (to reduce overfitting), normalization, early stopping

4. Models inspired from: Inception(GoogleNet), ViT (and transfered learning using pretrained GoogleNet models to do fine tuning if there's time)
